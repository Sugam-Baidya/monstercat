# Monstercat Project: Dependencies & Tools

This document outlines the key external libraries and tools used in the Monstercat project.

## Carousel Library

The project utilizes **Embla Carousel**, a lightweight, performant, and highly customizable carousel library. It is integrated into the project using the following packages:

*   **`embla-carousel-react`**: The official React wrapper for Embla Carousel. It provides the `useEmblaCarousel` hook for seamless integration into React components.
*   **`embla-carousel-autoplay`**: A plugin for Embla Carousel that adds autoplay functionality, allowing the carousel to advance automatically.

Embla is chosen for its "bring your own styles and markup" approach, giving developers full control over the carousel's appearance and behavior.

## Other External Tools

Below is a list of other significant external libraries and tools that are foundational to this project.

### Library & UI

*   **React**: The core of the application is built with React, the industry-standard JavaScript library for building dynamic user interfaces.
*   **Framer Motion**: A powerful and easy-to-use animation library for React. It is likely used for creating fluid UI animations and transitions.
*   **Lucide React**: A library providing a set of simple, clean, and customizable SVG icons.

### Styling

*   **Tailwind CSS**: A utility-first CSS framework that allows for rapid UI development by composing utility classes directly in the markup.

### Development & Build Tools

*   **Vite**: A modern frontend build tool that provides an extremely fast development server and bundles the application for production.
*   **TypeScript**: A statically typed superset of JavaScript that enhances code quality, readability, and developer productivity by catching errors during development.
*   **ESLint**: A pluggable linter tool for statically analyzing code to quickly find and fix problems, ensuring code consistency and quality.
